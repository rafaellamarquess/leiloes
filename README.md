# LeiloesTDSat
Projeto criado para uma atividade do curso de programação do SENAC. O objetivo do projeto é aprender sobre vercionamento.
A atividade possui um projeto em Java e um banco de dados SQL. O banco de dados foi adicionado a pasta do projeto Java e
foram criados dois "commits" registrando a criação do projeto e a adição do arquivo SQL. Em seguida o projeto foi adicionado ao GitHub.
No GitHub foi adicionado um "Readme" ao projeto, depois esse Readme foi adicionado a pasta local do projeto.

## Tecnologias utilizadas
- Java
- MySQL
